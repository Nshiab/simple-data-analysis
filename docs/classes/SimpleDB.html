Class SimpleDB

SimpleDB is a class that provides a simplified interface for working with DuckDB, a high-performance, in-memory analytical database. This class is meant to be used in a web browser. For NodeJS and other runtimes, use SimpleNodeDB.

Here's how to instantiate and start a SimpleDB instance.

const sdb = await new SimpleDB().start()
